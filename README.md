# Conversor_de_Videos 
App web em Node.js e React

![Imagem do Projeto](./front/conversor/src/Images/Conversor.png)

#Colocando em Funcionamento

Instale as dependencias
```
$ npm install
```
Inicie o servidor node
```
$ cd back
```
```
$ node server.js
```
Em outro terminal, inicie o react:
```
$ cd front/converter
```
```
$ npm start
```
